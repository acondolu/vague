# Type System

## Primitive types

- `bool`, `byte`, `int`, `uint`, `float`
- `u8`, `u16`, `u32`, `u64`, `u128`, `i8`, `i16`, `i32`, `i64` and `i128`
- `f32`, `f64`

## Special types
- `any`

## Builtins
- `bytes`: an array of bytes
- `char`: a UTF-8 Unicode code point (one to four bytes)
- `string`: a UTF-8 Unicode string
